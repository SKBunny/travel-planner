# ✈️ Travel Planner 
Веб-сайт для планування подорожей, який об'єднує маршрути, активності, бюджет, нотатки та список речей в одному інтерфейсі.

## 📌 Опис проєкту

**Travel Planner** — це система, яка допомагає користувачам повністю організувати подорож:

* створювати поїздки
* планувати маршрут
* додавати активності
* контролювати бюджет
* зберігати нотатки
* вести список речей

Мета проєкту — замінити кілька окремих сервісів одним зручним безкоштовним рішенням.


## 🚀 Основний функціонал

### 🔐 Аутентифікація

* Реєстрація та вхід користувачів
* Персональні дані та ізоляція інформації


### 🧳 Управління поїздками

* Створення, редагування та видалення поїздок
* Вказання:

  * назви
  * країни/міста
  * дат
  * бюджету

### 🗺️ Маршрути та транспорт

* Додавання маршрутів між містами
* Підтримка типів транспорту:

  * ✈️ літак
  * 🚆 потяг
  * 🚌 автобус
  * 🚗 авто


### 📅 Календар

* Відображення поїздок по днях

### 🎯 Активності

* Додавання активностей до поїздки
* Поля:

  * назва
  * дата і час
  * місце
  * опис
  * вартість

### 💰 Бюджет

* Загальний бюджет поїздки
* Витрати по активностях
* Автоматичний підрахунок:

  * витрачено
  * залишок


### 📝 Нотатки

* Додавання нотаток до поїздки
* Категорії:
  * загальне
  * важливе
  * контакти
  * посилання
* Закріплення нотаток


### 🎒 Packing List

* Список речей
* Категорії:

  * одяг
  * засоби особистої гігієни 
  * електроніка
  * документи
  * інше
* Позначення зібраних речей


### 🌍 Карта світу

* Позначення країн:

  * відвідано
  * заплановано
* Додавання нотаток до країн


### 🤖 AI Асистент

* Чат для отримання рекомендацій
* Допомога у плануванні маршруту
* Поради щодо подорожей

### 🔍 Глобальний пошук

* Пошук по:

  * поїздках
  * активностях
  * нотатках
  * готелях
* Пошук по ключових словах (не тільки по повній фразі)


### 📄 Експорт у PDF

* Генерація повного плану поїздки
* Включає:

  * маршрут
  * активності
  * бюджет
  * список речей

## 📂 Структура проєкту
